## Funis de campanhas de tráfego pago

Aplicação web para criação visual de funis de campanhas de tráfego pago.

O usuário pode criar etapas do funil (como anúncio, landing page, formulário, checkout), conectar essas etapas visualmente e visualizar o fluxo completo de forma gráfica.

Cada etapa apresenta dados simulados de acessos e conversões apenas para demonstrar como seria o desempenho do funil.

## Tecnologias utilizadas

- React
- React Flow
- shadcn/ui
- Tailwind CSS
- Vite

## Funcionalidades

- Criação de etapas do funil
- Conexão visual entre etapas
- Visualização gráfica do fluxo
- Dados simulados de acessos e conversões
- Persistência local utilizando `localStorage`

## Como executar o projeto

Instale as dependências:

npm install

Execute o projeto:

npm run dev
